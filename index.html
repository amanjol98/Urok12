Наша сегодняшняя тема это свойтсва CSS

CSS (Cascading Style Sheets) предоставляет множество свойств, которые можно использовать для стилизации HTML-элементов.

Для примера сейчас создаем HTML-документ с несколькими элементами текста (заголовки, абзацы).

Чтобы добавить стили для следующих текстовых свойств: color, font-size, text-align, font-weight

Контейнер, где я применил бэкграунд колор и бэкграунд имедж для фона
Создайте несколько элементов с разными размерами
- `width` и `height` — задайте ширину и высоту для элементов.
- `overflow` — установите управление содержимым, если оно выходит за границы блока (например, `hidden`, `scroll`).
